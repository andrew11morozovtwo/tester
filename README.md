### README.md для Проекта Тестирования Платформы idemo.bspb.ru

---

## О проекте

Этот проект представляет собой комплексное тестирование платформы idemo.bspb.ru, выполненное в рамках итогового проекта по курсу тестирования программного обеспечения. Проект включает в себя ручное тестирование, создание тестовой документации и анализ результатов тестирования.

### Функционал

- **Майнд-карты**: Созданы майнд-карты жизненного цикла тестирования ПО и методологии разработки ПО.
- **Тестовая документация**: Разработаны чек-листы, тест-кейсы и баг-репорты для ручного тестирования платформы.
- **Техники тест-дизайна**: Применены техники тест-дизайна для создания эффективных тест-кейсов.
- **Автотестирование**: Планировалось выполнить автотестирование с помощью Selenium, но не было реализовано из-за ограничений по времени.
- **Анализ результатов**: Выявлены недостатки платформы и даны рекомендации по улучшению.

### Используемые инструменты

- **Selenium**: Планировался для автотестирования.
- **Тестовая документация**: Чек-листы, тест-кейсы, баг-репорты.
- **Платформа**: idemo.bspb.ru

### Результаты

- **Ручное тестирование**: Выявлены баги и уязвимости платформы.
- **Рекомендации**: Платформа не рекомендована к релизу без устранения выявленных проблем.

### Выводы

Проект показал важность ручного тестирования на данном этапе разработки платформы. Рекомендуется продолжить работу над устранением багов и реализацией автоматизированного тестирования.
